## 영화 추천 알고리즘

- 최신 유행 영화 추천
    - TMDB 기준 오늘 유행하는 영화 20개 추천

- 사용자 맞춤 영화 추천
    - 사용자가 많이 본 영화 장르에 기반해 추천
    1. 리뷰 작성한 영화를 기준으로 상위 장르들을 선정
    2. DB 내에서 유사 장르들의 영화를 선정
    3. 선정한 영화들의 평점을 기준으로 상위 20개 영화 추천
